# TrackIT: Events, Documents, and Tasks Tracker System

A UI/UX design project for a unified tracker system that manages events, documents, and tasks.

## Project Description
TrackIT is a system designed to help users organize and monitor their events, documents, and tasks in one place. This repository contains the design assets, style guide, and project documentation.
