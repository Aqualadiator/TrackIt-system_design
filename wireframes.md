# Wireframes

## Screens Overview
- Dashboard — shows summary of events, documents, and tasks
- Events Page — list and calendar view of upcoming events
- Documents Page — file list with upload and search functionality
- Tasks Page — task list with status indicators (pending, in progress, done)
